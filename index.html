常见问题

FAQ

FB 户的 4%（3+1）是什么意思？
4% 是充值手续费 —— 你充 $100 到广告账户，实际支付 $104。
(3+1) 表示 3 美金平台费 + 1 美金服务费，每笔充值都包含进去。
TK 户的 5.5%（4.5+1）同理。
账户多久能到手？
完成付款 + 确认配置后，当日内交付。
如果是批量订单（10 户以上）或特殊地区户型，客服会提前告知交付时间，一般 1-2 个工作日。
账户被封了怎么办？
在保障期内被封，把账户信息和被封截图发给客服。
验证封号原因（非违规投放导致）后，免费补发同等级账户。
建议配合 ipcloak.ai Browser + Cloak 使用，把账户存活率拉到最高。
支持哪些支付方式？
支持 USDT、对公汇款、PayPal 等多种渠道。
具体支付方式请直接跟 TG 客服 @webpage1688 沟通。
账户的时区和地区可以自选吗？
可以。下单前请告知客服你需要的：
· 时区（美区 / 欧洲 / 东南亚 / 港台）
· 投放地区（你的目标市场）
· 行业类别（电商 / 应用 / 内容 / 金融 / 灰产边缘）
客服会匹配最合适的户型。
BM 和粉丝页是必须配的吗？
不是必须，但强烈推荐。
· BM 是 Facebook 投放的资产容器，分离账户和资产，关联封禁时只影响单一账户而不会牵连其他资源。
· 粉丝页是广告必备载体，自己养的粉丝页质量参差不齐，配套购买的真实养护粉丝页可以提升广告通过率。
跟你们其他产品（Browser / Cloak）有什么关系？
账户 + 工具 = 完整投放链路，三者协同效果最佳：
· ipcloak.ai Browser —— 每个账户独立指纹隔离，避免硬件关联封禁
· ipcloak.ai Cloak —— 广告斗篷，过审核 IP + 落地页分流
· ipcloak.ai Accounts —— 账户本身 + BM + 粉丝页全套资源
买齐才是出海广告完整工具链。
跟 ads.ipcloak.ai 有什么区别？
· accounts.ipcloak.ai（本站）—— 明码标价的账户零售，自己有投手能力的客户买户自投。
· ads.ipcloak.ai —— 开户 + 代投托管，包含投手团队和优化服务，适合不想自己跑广告的客户。
